# bible-cli

Clone, run the installer, and use the `bible` command.

Clone the repo:

```
git clone https://github.com/dionjoshualobo/bible-cli.git
cd bible-cli
```

Run the installer:

```
./install.sh
```

If the installer is not executable, make it so:

```
chmod +x ./install.sh
./install.sh
```

After installation run the `bible` command:

```
bible
```

Enjoy!
